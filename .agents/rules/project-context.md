# Project Context: my-careerpath

## 1. Project Overview
**my-careerpath** is an AI-driven career growth platform. The application allows job seekers to upload their PDF CVs, which are then analyzed by an AI agent to identify skill gaps and generate a career growth trajectory. A secondary sub-agent utilizes search tools to find real-world company matches based on the generated career plan.

## 2. Architecture & Tech Stack
The project uses a decoupled architecture to separate the heavy AI processing from the clean frontend experience.

### Frontend (UI/UX)
* **Framework:** Next.js 15 (App Router)
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Core Libraries:** `react-dropzone` (for PDF upload), `lucide-react` (for iconography).

### Backend (AI Logic & API)
* **Framework:** Python with FastAPI
* **AI Orchestration:** Google Agent Development Kit (ADK)
* **Primary LLM:** Gemini 3.1 Pro
* **Tools:** `GoogleSearchTool` (for the scout agent)
* **PDF Processing:** `PyMuPDF` or `pdfplumber`

## 3. Strict UI/UX Guidelines (The "Cinematic SaaS" Standard)
All frontend components MUST adhere strictly to the following design system to ensure a premium, modern feel:
* **Aesthetic:** Clean, modern SaaS minimalism. No clutter.
* **Color Palette:** Strictly neutral tones. 
  * Background: `#fafafa`
  * Text Primary: `#18181b`
  * Accents: Subtle Zinc or Slate variations. No harsh or overly vibrant colors.
* **Lighting & Depth:** Apply a "soft neutral lighting" effect to UI elements. Use very diffuse, soft shadows and avoid harsh borders. Modal overlays should utilize a subtle background blur to mimic a shallow depth of field.
* **Motion & Transitions:** All animations (hover states, modal openings, page transitions, loading states) must be smooth, slow, and continuous. Use Framer Motion with durations between `0.5s` and `0.8s` and gentle easing curves to create a cinematic, high-end feel.
* **Typography:** Inter or Geist, prioritizing readability and ample whitespace.

## 4. Multi-Agent Workflow Blueprint

### Agent 1: The Analyzer (Main Agent)
* **Role:** Parse extracted PDF text, analyze current skills, and build a structured 3-step career growth plan.
* **Input:** Raw text from CV.
* **Output:** JSON mapping of `current_skills`, `skill_gaps`, and `milestones`.

### Agent 2: The Scout (Sub-Agent)
* **Role:** Take the career profile generated by the Analyzer and execute targeted searches to find matching companies.
* **Tools:** `GoogleSearchTool`
* **Input:** Career trajectory data.
* **Output:** Array of 3 recommended companies with name, brief rationale, and potential roles.
